ergPost für e107 v0.7+
======================

Verzeichnis "ergpost" unter e107_plugins speichern.
Im selben Verzeichnis muss sich der LMO 4.0.x befinden, hier die im Paket mitgelieferten Dateien aus dem Verzeichnis "lmo" überspielen.

ergPost kann über das Admin-Backend als Plugin installiert und administriert werden.

Zwingend im Plugin als erstes die Installation aufrufen!
